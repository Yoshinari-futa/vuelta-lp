### ステップ2: Vercelでデプロイ

#### 方法A: Vercelダッシュボードから（推奨・最も簡単）

1. **Vercelにアクセス**
   - https://vercel.com を開く
   - GitHubアカウントでログイン

2. **プロジェクトをインポート**
   - 「Add New...」→「Project」をクリック
   - 「Import Git Repository」で `Yoshinari-futa/vuelta-lp` を検索して選択
   - または、GitHubリポジトリのURLを直接入力

3. **設定を確認（自動検出されるはず）**
   - Framework Preset: **Next.js**
   - Root Directory: `./`
   - Build Command: `npm run build`
   - Output Directory: `.next`
   - Install Command: `npm install`

4. **環境変数（現在は不要）**
   - チャットボックスは削除済みのため、環境変数の設定は不要です

5. **デプロイ開始**
   - 「Deploy」ボタンをクリック
   - 2-3分待つとデプロイが完了します
   - 完了すると、`https://vuelta-lp-xxxxx.vercel.app` のようなURLが表示されます

## 🔧 トラブルシューティング

### ビルドエラーが発生する場合
1. Vercelのビルドログを確認
2. エラーメッセージを確認
3. 必要に応じてローカルで `npm run build` を実行して確認

### フォントが読み込まれない場合
- Google Fonts（Playfair Display, Inter）はVercelのビルド環境で自動的に読み込まれます
- カスタムフォント（`これ.ttf`, `BTAnnamNeue-Regular.otf`）は `app/fonts/` に配置されていることを確認

### 画像が表示されない場合
- `public/images/` ディレクトリ内の画像ファイルが正しく配置されているか確認
- Unsplash画像は `next.config.js` でリモートパターンが設定済み

## 📝 カスタムドメインの設定（オプション）

デプロイ後、カスタムドメインを設定する場合：

1. Vercelダッシュボードでプロジェクトを選択
2. Settings → Domains
3. ドメインを追加（例: `vuelta-bar.com`）
4. DNS設定を案内に従って設定
