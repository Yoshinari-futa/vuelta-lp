# ローカルで確認する方法

## 方法1: 本番ビルドで確認（推奨・安定）

```bash
cd /Users/yoshinarifuta/lp
npm run build
npm run start
```

ブラウザで **http://localhost:3000** を開いてください。

## 方法2: 一括コマンド

```bash
cd /Users/yoshinarifuta/lp
npm run preview
```

ビルド後に自動でサーバーが起動します。

## 確認URL

- 英語版: http://localhost:3000
- 日本語版: http://localhost:3000/ja
- 採用ページ: http://localhost:3000/recruit

## 注意

- 開発サーバー（`npm run dev`）で「EMFILE: too many open files」が出る場合は、上記の本番サーバー（`npm run start`）を使ってください
- サーバーを止めるには、ターミナルで `Ctrl + C` を押してください
