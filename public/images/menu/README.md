# メニュー画像

このディレクトリにメニュー画像を配置してください。

## ファイル名

以下のファイル名で画像を配置してください：

- `menu-page-01.jpg` - 1ページ目（VUELTA SIGNATURES）
- `menu-page-02.jpg` - 2ページ目（CLASSIFIED COMMODITIES）
- `menu-page-03.jpg` - 3ページ目（GLOBAL CURRENCY）
- `menu-page-04.jpg` - 4ページ目（INVESTMENT GRADE）

## 推奨形式

- **形式**: JPG, PNG, WebP
- **サイズ**: 推奨 1200x1600px（3:4のアスペクト比）
- **ファイルサイズ**: 各画像500KB以下を推奨

## 使用方法

画像を配置したら、フルメニューボタンをクリックするとモーダルで表示されます。
左右の矢印ボタンまたはキーボードの矢印キーでページを切り替えられます。
