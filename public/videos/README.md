# 動画ファイルの配置

## 📁 動画ファイルの配置場所

動画ファイルをこのフォルダ（`public/videos/`）に配置してください。

## 🎥 必要な動画ファイル

### Heroセクションの背景動画
- **ファイル名**: `hero-background.mp4`
- **推奨サイズ**: 1920x1080px (Full HD)
- **ファイルサイズ**: 5MB以下を推奨
- **長さ**: 15-30秒のループ動画
- **形式**: MP4 (H.264)
- **音声**: サイレント（音声なし）推奨

## 📸 ポスター画像（オプション）

動画の最初のフレームを画像として保存し、`public/images/video-poster.jpg` に配置してください。
動画が読み込まれるまでの間、この画像が表示されます。

## ⚙️ 動画の最適化

動画ファイルが大きい場合は、以下のツールで最適化してください：

### FFmpegを使用する場合
```bash
# MP4に変換・圧縮
ffmpeg -i input.mov -vcodec h264 -acodec aac -crf 23 -preset slow -movflags +faststart hero-background.mp4
```

### オンラインツール
- HandBrake
- CloudConvert
- FreeConvert

## 📝 注意事項

1. **ファイルサイズ**: 大きな動画ファイルは読み込みが遅くなります。必ず最適化してください。
2. **モバイル対応**: 動画は自動的にモバイルでも再生されますが、データ使用量に注意してください。
3. **パフォーマンス**: 動画が読み込めない場合は、フォールバック背景が表示されます。

## 🚀 次のステップ

1. 動画ファイルを `hero-background.mp4` としてこのフォルダに配置
2. （オプション）ポスター画像を `public/images/video-poster.jpg` に配置
3. ブラウザで確認
