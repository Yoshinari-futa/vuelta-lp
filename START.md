# サイトを表示する方法

## 🚀 簡単な手順

### 1. ターミナルを開く

Macの場合は：
- `Cmd + Space` でSpotlight検索を開く
- 「ターミナル」または「Terminal」と入力してEnter

### 2. プロジェクトディレクトリに移動

ターミナルで以下を入力してEnterキーを押してください：

```bash
cd ~/lp
```

### 3. 開発サーバーを起動

以下のコマンドを入力してEnterキーを押してください：

```bash
npm run dev
```

### 4. ブラウザで開く

ターミナルに以下のようなメッセージが表示されます：

```
▲ Next.js 14.2.35
- Local:        http://localhost:3000
✓ Ready in 2.3s
```

**このURL（`http://localhost:3000`）をブラウザで開いてください。**

## 📝 手順の詳細

1. **ターミナルを開く**
   - Mac: `Cmd + Space` → 「ターミナル」と入力
   - Windows: `Win + R` → 「cmd」と入力

2. **プロジェクトディレクトリに移動**
   ```bash
   cd ~/lp
   ```
   または
   ```bash
   cd /Users/yoshinarifuta/lp
   ```

3. **開発サーバーを起動**
   ```bash
   npm run dev
   ```

4. **ブラウザで開く**
   - ターミナルに表示されたURLをコピー
   - ブラウザのアドレスバーに貼り付けてEnter

## ⚠️ 注意事項

- 開発サーバーを起動したら、**ターミナルを閉じないでください**
- サーバーを停止する場合は、ターミナルで `Ctrl + C` を押してください
- ポート番号が3000以外の場合（例：3001、3002など）は、その番号のURLを開いてください

## 🔄 変更を反映するには

コードを変更したら、ブラウザをリロード（F5またはCmd+R）すると自動的に反映されます。
