# Google Maps Setup

## 現在の設定

Google Mapsの埋め込みは設定済みですが、正確な座標を使用するために、以下の手順で埋め込みURLを更新してください。

## 手順

1. Google Mapsで「広島市中区大手町3丁目3-5 掛江ビル2F」を検索
2. 「共有」ボタンをクリック
3. 「地図を埋め込む」タブを選択
4. 表示されたHTMLコードから`src`属性のURLをコピー
5. `app/page.tsx`の`<iframe>`タグの`src`属性を更新

## 現在のURL

現在は検索クエリベースのURLを使用していますが、正確な座標を使用するために上記の手順で更新することを推奨します。

## 代替方法

もし埋め込みマップが表示されない場合、以下のリンクでGoogle Mapsを開くことができます：

```
https://www.google.com/maps/search/?api=1&query=広島市中区大手町3丁目3-5+掛江ビル2F
```
