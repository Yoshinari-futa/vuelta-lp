## 🚀 公開方法（最も簡単）

### 方法1: Vercelダッシュボードから直接デプロイ（推奨・最も簡単）

1. **Vercelにアクセス**
   - https://vercel.com を開く
   - GitHubアカウントでログイン（なければ作成）

2. **プロジェクトをインポート**
   - ダッシュボードで「Add New...」→「Project」をクリック
   - 「Import Git Repository」で `Yoshinari-futa/vuelta-lp` を検索して選択

3. **設定を確認**
   - Framework Preset: **Next.js**（自動検出されるはず）
   - Root Directory: `./`（そのまま）
   - Build Command: `npm run build`（自動）
   - Output Directory: `.next`（自動）
   - その他はデフォルトのまま

4. **デプロイ開始**
   - 「Deploy」ボタンをクリック
   - 2-3分待つとデプロイが完了します
   - 完了すると、`https://vuelta-lp-xxxxx.vercel.app` のようなURLが表示されます

5. **完了！**
   - 表示されたURLにアクセスして確認してください

### 方法2: GitHubにプッシュしてからVercelで連携

もしGitHubにプッシュしたい場合：

1. **GitHubにプッシュ**
   ```bash
   # ターミナルで実行
   cd /Users/yoshinarifuta/lp
   git push origin main
   ```
   - 認証が求められたら、GitHubのPersonal Access Tokenを使用
   - または、GitHub Desktopなどのアプリを使用

2. **Vercelで連携**
   - 上記の方法1と同じ手順で、VercelダッシュボードからGitHubリポジトリを連携
   - 今後は、GitHubにプッシュするだけで自動的にデプロイされます

## 📝 注意事項

- **フォントファイル**: カスタムフォント（`これ.ttf`）は既にリポジトリに含まれているので、Vercelでも正しく読み込まれます
- **環境変数**: 現在チャットボックスは削除されているため、環境変数の設定は不要です
- **カスタムドメイン**: 後で設定したい場合は、Vercelダッシュボードの「Settings」→「Domains」から設定できます
