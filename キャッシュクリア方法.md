# ブラウザのキャッシュをクリアする方法

## 方法1: ハードリロード（最も簡単）

### Mac
- **Chrome/Safari/Firefox**: `Cmd + Shift + R`
- **または**: `Cmd + Option + R`

### Windows
- **Chrome/Firefox**: `Ctrl + Shift + R`
- **または**: `Ctrl + F5`

## 方法2: 開発者ツールからキャッシュをクリア

1. ブラウザで `F12` または `Cmd + Option + I` (Mac) / `Ctrl + Shift + I` (Windows) を押して開発者ツールを開く
2. ネットワークタブを開く
3. 「Disable cache」にチェックを入れる
4. ページをリロード

## 方法3: ブラウザの設定からキャッシュをクリア

### Chrome
1. `Cmd + Shift + Delete` (Mac) / `Ctrl + Shift + Delete` (Windows)
2. 「キャッシュされた画像とファイル」を選択
3. 「データを削除」をクリック

### Safari
1. `Cmd + Option + E` でキャッシュをクリア
2. または、Safari → 環境設定 → 詳細 → 「メニューバーに"開発"メニューを表示」にチェック
3. 開発メニュー → 「キャッシュを空にする」

### Firefox
1. `Cmd + Shift + Delete` (Mac) / `Ctrl + Shift + Delete` (Windows)
2. 「キャッシュ」を選択
3. 「今すぐ消去」をクリック

## 方法4: シークレット/プライベートモードで開く

キャッシュを使わずにページを開く：
- **Chrome**: `Cmd + Shift + N` (Mac) / `Ctrl + Shift + N` (Windows)
- **Safari**: `Cmd + Shift + N` (Mac)
- **Firefox**: `Cmd + Shift + P` (Mac) / `Ctrl + Shift + P` (Windows)

その後、http://localhost:3000 にアクセス

---

**推奨**: 方法1（ハードリロード）が最も簡単です！
