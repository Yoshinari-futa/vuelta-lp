# 🚀 今すぐ公開する方法（3ステップ）

## ステップ1: GitHubにプッシュ

ターミナルで以下を実行してください：

```bash
cd /Users/yoshinarifuta/lp
git add .
git commit -m "公開前の最終調整: SEO/MEO対策、モバイル最適化完了"
git push origin main
```

**注意**: GitHubの認証が必要な場合：
- Personal Access Tokenを使用する
- または、GitHub Desktopなどのアプリを使用

## ステップ2: Vercelでデプロイ

1. **Vercelにアクセス**
   - ブラウザで https://vercel.com を開く
   - GitHubアカウントでログイン（なければ作成）

2. **プロジェクトをインポート**
   - ダッシュボードで「Add New...」→「Project」をクリック
   - 「Import Git Repository」をクリック
   - `Yoshinari-futa/vuelta-lp` を検索して選択
   - または、リポジトリURLを直接入力: `https://github.com/Yoshinari-futa/vuelta-lp`

3. **設定を確認（自動検出されるはず）**
   - Framework Preset: **Next.js** ✅
   - Root Directory: `./` ✅
   - Build Command: `npm run build` ✅
   - Output Directory: `.next` ✅

4. **デプロイ開始**
   - 「Deploy」ボタンをクリック
   - 2-3分待つとデプロイが完了します

## ステップ3: 確認

デプロイが完了すると、以下のようなURLが表示されます：
- `https://vuelta-lp-xxxxx.vercel.app`

このURLにアクセスして、以下を確認してください：
- ✅ ホームページが表示される
- ✅ 日本語版（/ja）が表示される
- ✅ リクルートページ（/recruit）が表示される
- ✅ モバイル表示が正常に動作する

## 🎉 完了！

これで公開完了です！

## 📝 カスタムドメインの設定（オプション）

後でカスタムドメイン（例: vuelta-bar.com）を設定する場合：
1. Vercelダッシュボードでプロジェクトを選択
2. Settings → Domains
3. ドメインを追加
4. DNS設定を案内に従って設定
